| Format specifier | Expected type |
| ---------------- | ------------- |
| `%s`             | `*[i8]`       |
| `%#`             | `*:0[i8]`     |
| `%@`             | `*i8`         |
| `%c`             | `i8` (ASCII)  |
| `%b`             | `i8` (number) |
| `%s`             | `i16`         |
| `%i`             | `i32`         |
| `%u8`            | `i32` (UTF-8) |
| `%l`             | `i64`         |
| `%f`             | `f32`         |
| `%d`             | `f64`         |
|                  |               |
